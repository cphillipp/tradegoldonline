Easy to use Wordpress plugin to insert any Ads to your posts and BuddyPress sections.

== Description ==

Use this plugin to quickly and easily insert Any Ad code Unit to your posts and BuddyPress sections including Forum topics.


== Installation ==

1. Unzip and Upload the folder 'ads-manager' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings and configure your Ads code in the options page.
